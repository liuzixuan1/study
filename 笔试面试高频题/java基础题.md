1、Java的基本数据类型与包装类

基本类型：int,  short,	long ,	byte,	double,	float,	boolean,	char

| 基本数据类型 | 包装类  | 位数 | 默认值 |
| ------------ | ------- | ---- | ------ |
| byte         | Byte    | 8    | 0      |
| short        | Short   | 16   | 0      |
| int          | Integer | 32   | 0      |
| long         | Long    | 64   | 0L     |
| float        | Float   | 32   | 0f     |
| double       | Double  | 64   | 0d     |
| char         | Char    | 16   | 'u000' |
| boolean      | Boolean | 1    | false  |
